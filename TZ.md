Агрегатор почт.

Необходимо реализовать сервис для добавление почтовых аккаунтов с разных доменов с возможностью чтения всех сообщений в одном месте.

1. Вход в сервис будет осуществляться только с помощью админки. В админ панели админ может добавлять нового пользователя, добавив логин.
2. После того как админ создал нового пользователя, пользователь может войти в сервис с логином и придумать себе пароль.
3. В админ панели должны отображаться списки всех пользователей, список почт привязанных к этому пользователю а так же две кнопки "Сброс" которая будет сбрасывать пароль пользователя, "Удалить", которая будет удалять пользователя
4. При входе в аккаунт пользователь сможет добавить почту, введя логин и пароль. Почт может быть несколько. Все сообщения со всех почт отображается в одном месте
5. Пользователь может так же отвечать на сообщения с данного сервиса.
6. Необходимо реализовать пулинг каждые n секунд для синхронизации новых сообщений в агрегатор
